# Numberstring 

The `numberstring` package can translate integers into strings in python. 


## Installation 

You can install this via `pip`. 

```
pip install numberstring
```

## Usage 

```python
import numberstring 

numberstring.translate(1) # "one"
numberstring.translate(101) # "one-hundred-and-one"
```
